## Features

- Shared/global dashboard views
- Icinga DB-oriented widgets and layouts
- Consistent dashboards across teams

## Requirements

- Icinga Web 2
- Icinga DB stack enabled
- Compatible module dependencies

## Installation

```bash
git clone https://github.com/DanielVd/icingaweb2-module-global-dashboards-for-icingadb.git
```

Place module in Icinga Web 2 modules directory and enable it.

## Configuration

Configure module settings from Icinga Web 2 module administration.

## Usage

Open dashboards section in Icinga Web 2 and assign global dashboards as needed.

## Troubleshooting

- Module not visible: verify module path and enable state
- Empty widgets: verify Icinga DB backend connectivity
